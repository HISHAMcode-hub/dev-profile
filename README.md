## Setup

```bash
git clone https://github.com/muhammed-hisham/dev-profile.git
cd dev-profile
open index.html
```

Or just double-click `index.html` — no install, no npm, nothing.

## File Structure

```
dev-profile/
├── index.html     ← page content and structure
├── style.css      ← all styling
└── README.md      ← this file
```

## Deploy to GitHub Pages

1. Push repo to GitHub as `dev-profile`
2. Go to **Settings → Pages**
3. Source → **Deploy from a branch**
4. Branch → **main** | Folder → **/ (root)**
5. Click **Save** — live in ~60 seconds
